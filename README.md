
# Simple Interest Calculator

This project contains a Bash script that calculates simple interest.

## Formula Used
Simple Interest = (Principal × Rate × Time) / 100

## How to Run

1. Open terminal
2. Run the script:
   bash simple-interest.sh

3. Enter:
   - Principal amount
   - Rate of interest
   - Time period

The script will calculate and display the simple interest.
